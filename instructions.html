    Instructions

    Installation

    This android app is an extension to WebSMS. So

    • First, if you don't already have it, install WebSMS from Google Play here.
    • Then, install the WebSMS Cabbage Connector from Google Play here.

    When you start WebSMS the next time, you'll be able to configure the Cabbage Connector as described below. Note that you must configure it with your account details before you can use it.

    Configuration

    To configure WebSMS Cabbage Connector, you first need to get to its preferences screen. For this,

    1. Start WebSMS
    2. Press Menu
    3. Select "Preferences" / "Preferences - cabbage"

    Enabling the Connector

    All connectors are installed in the disabled mode. You must enable the Connector to use it. To do this,

    1. Go to the Connector's preferences screen (see above)
    2. Tick "Enable cabbage"

    NOTE that you need to add at least one account (as described below) to use this Connector.

    Adding an account

    1. Go to the Connector's preferences screen (see above)
    2. Press Menu
    3. Select "Add Account"
    4. Populate all fields (see Account details explained)
    5. Press Back to return to the main preferences screen

    Modifying account details

    1. Go to the Connector's preferences screen (see above)
    2. Select account to modify
    3. Update account details (see Account details explained)
    4. Press Back to return to the main preferences screen

    Deleting an account

    1. Go to the Connector's preferences screen (see above)
    2. Select account to delete
    3. Press Menu
    4. Select "Delete Account"
    5. On the confirmation popup "Delete this account?", select "Delete"

    Account details explained

    • Account Name  - type in a small unique label for this account record
    • Provider  - select your mobile provider from the list
    • Custom Provider  - (advanced) see TODO
    • User Name  - user name that you use to login to the mobile provider's web site
    • Password  - password that you use to login to the mobile provider's web site
