Support

If you have a question about this application or experience some problem, first of all, have a look at the FAQ section. It might already be addressed there.

Issues

I am happy to help you with resolving any issues related to this application. Note, however, that there are quite a few components involved here. So, if you have some difficulties with sending your message, it does not necessarily mean that the problem is with the application. To localise the problem, please follow the steps below.

  1. First, check that you can send messages on the mobile provider's web site. It does happen once in a while that these sites go down for various reasons. If you experience problems there, you can contact your mobile provider's customer support.
  2. Secondly, check that Cabbage Gateway works fine. The best way to verify this is to send a message via their Web based version. Sometimes, mobile providers change their web sites and it takes some time for the Cabbage to catch up. If you experience problems there, you can contact Cabbage developer on his Feedback page.
  3. If all the above works, then the problem might indeed be with this application. Please record the issue or question on the WebSMS's Issues page and I'll look after it.
