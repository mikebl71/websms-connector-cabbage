This is WebSMS Connector for Cabbage Gateway. It's an android app that lets you send free/cheap sms messages via web sites of the following Irish mobile providers: Vodafone, Meteor, Three, Tesco Mobile, eMobile, and also of O2 UK, Telenor Serbia.

Note that, in order to use this application, you must be a customer of one of the mobile providers listed above and be registered on their web site.

This app is an extension to WebSMS, so you also need to install WebSMS before you can use this app.

Overview

Most Irish mobile providers allow sending some sms messages from their web sites for free. (Obviously, you need to have a phone from this provider is order to use this service.) However, it is a bit of a pain that you need to use a computer and login to their web site every time you want to send an sms. So some mobile apps were developed that let you access those free web texts directly from your mobile. There is one problem here, however. Provider's web sites are not optimised for mobile usage and are rather verbose. So if you pay for mobile data traffic per kilobyte (which you do on most Pay As You Go plans or while roaming) then you still end up consuming (and paying for) more mobile data than necessary.

The alternative is to use the Cabbage Gateway. This is a web service that was developed by one smart Irish guy and that is specifically optimised for mobiles. It keeps communication with mobile phone to absolute minimum. So you really only pay for the size of your message (a small fraction of a kilobyte). The rest of the talking with the provider's web site is done by the gateway on your behalf.

There are several ways to use Cabbage Gateway from your mobile. This "WebSMS Cabbage Connector" was developed to allow you to send free/cheap sms messages via the gateway using the famous WebSMS android app. WebSMS is a powerful tool for sending free/cheap sms messages that is used by thousands of people worldwide. A number of connectors were developed to enable its users to use various sms web services available on the Internet. With the help of this Cabbage Connector, you can now use WebSMS to send sms messages via Cabbage Gateway as well.
